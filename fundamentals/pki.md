# 🔐 Public Key Infrastructure (PKI)

## 📌 What is PKI?

Public Key Infrastructure (PKI) is a system that manages **digital keys and certificates** to enable secure communication between users and systems.

PKI का मतलब है एक ऐसा system जो keys और certificates को manage करता है ताकि communication secure हो सके।

---

## 🔐 How PKI Works

Momo wants to send secure data to a company server.

- The server shares its public key through a certificate  
- Momo’s system verifies that certificate  
- A secure connection is established  

Now Momo can safely send sensitive data.

Momo का system पहले verify करता है कि सामने वाला server trusted है या नहीं, फिर communication शुरू होता है।

---

## ⚠️ When PKI Fails

If the system trusts a fake certificate:

- Momo may connect to an attacker instead of the real server  
- Sensitive data can be stolen  

अगर trust गलत जगह पर हो जाए, तो पूरी communication compromise हो सकती है।

---

## 🎯 Interview Tip

PKI is all about **managing trust using keys and certificates**.

---

## 🚀 Key Takeaways

- PKI manages keys and certificates  
- Enables secure communication  
- Builds trust between systems
