# 🛠️ Security Controls

## 📌 What are Security Controls?

Security controls are measures used to **protect systems, data, and resources**.

Security controls system को secure रखने के लिए use किए जाते हैं।

---

## 🧠 Understanding Through a Scenario

Momo’s company uses:

- Security policies → Administrative  
- Firewall → Technical  
- CCTV → Physical  

👉 All together protect the organization.

---

## 🔑 Types of Security Controls

### 🔹 Administrative Controls
- Policies  
- Training  
- Procedures  

### 🔹 Technical Controls
- Firewalls  
- Encryption  
- Antivirus  

### 🔹 Physical Controls
- Locks  
- CCTV  
- Security guards  

---

## 📊 Business Perspective

- Reduces risk  
- Ensures compliance  
- Protects sensitive data  

---

## ⚠️ Common Issues

- Missing controls  
- Poor implementation  
- Lack of monitoring  

---

## 🛡️ Real-World Relevance

- Every organization uses these  
- Required for audits  

---

## 🎯 Interview Tips

- Give 1 example of each type  
- Explain clearly  
- Keep it simple  

---

## 🚀 Key Takeaways

- Controls protect systems  
- Must be layered and combined

# ⚙️ Control Types (Preventive, Detective, Corrective)

## 📌 What are Control Types?

Control types define **how security controls work**.

---

## 🧠 Understanding Through a Scenario

Momo’s system has:

- Firewall → Prevents attack  
- Logs → Detects attack  
- Backup → Recovers data  

---

## 🔑 Types of Controls

### 🔹 Preventive Controls
- Stop attacks before they happen  
- Example: Firewall  

### 🔹 Detective Controls
- Identify attacks  
- Example: Logs, IDS  

### 🔹 Corrective Controls
- Fix damage after attack  
- Example: Backup restore  

---

## 📊 Business Perspective

- Prevent loss  
- Detect incidents early  
- Ensure recovery  

---

## ⚠️ Common Issues

- No detection system  
- Weak recovery plan  

---

## 🎯 Interview Tips

- Use simple examples  
- Explain flow: Prevent → Detect → Correct  

---

## 🚀 Key Takeaways

- All 3 types are important  
- Work together for full protection
