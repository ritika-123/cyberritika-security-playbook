# 🔐 Symmetric Encryption in Cybersecurity

## 📌 What is Symmetric Encryption?

Symmetric Encryption is a method where **the same key is used to encrypt and decrypt data**.

Symmetric Encryption का मतलब है कि data को lock और unlock करने के लिए एक ही key use होती है।

---

## 🔐 How Symmetric Encryption Works

Momo wants to send a confidential file to his teammate.

- He encrypts the file using a secret key  
- The teammate uses the same key to decrypt it  

If someone intercepts the file but doesn’t have the key, they cannot read it.

लेकिन अगर key किसी attacker के पास पहुँच जाए, तो वह data को आसानी से पढ़ सकता है।

---

## ⚠️ When Symmetric Encryption Fails

Momo shares the secret key over an insecure channel (like plain message).

A hacker intercepts the key and can now:
- Decrypt all messages  
- Access sensitive data  

अगर key secure तरीके से share नहीं की गई, तो पूरी security compromise हो सकती है।

---

## 🎯 Interview Tip

Symmetric encryption is **fast but has a key distribution problem**.

---

## 🚀 Key Takeaways

- Same key is used for encryption and decryption  
- Very fast and efficient  
- Main challenge is secure key sharing
