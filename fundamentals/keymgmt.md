# 🔐 Key Management & Key Exchange

## 📌 What is Key Management?

Key Management is the process of **creating, storing, distributing, and protecting cryptographic keys**.

Key Management का मतलब है keys को securely manage करना।

---

## 📌 What is Key Exchange?

Key Exchange is the process of **securely sharing keys between two parties**.

---

## 🔐 How It Works

Momo wants to communicate securely.

- A key is generated  
- It is securely shared with the receiver  
- Both use it for encryption/decryption  

अगर key safe है, तो communication भी safe रहेगा।

---

## ⚠️ When It Fails

If keys are:

- Stored insecurely  
- Shared over unsafe channels  

Then attackers can:
- Steal keys  
- Decrypt data  

अगर key leak हो जाए, तो पूरी security खत्म हो जाती है।

---

## 🎯 Interview Tip

Security depends heavily on **how keys are managed, not just encryption algorithms**.

---

## 🚀 Key Takeaways

- Keys must be protected  
- Secure key exchange is critical  
- Weak key management = system compromise
