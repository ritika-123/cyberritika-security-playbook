# 🔐 Hashing in Cybersecurity

## 📌 What is Hashing?

Hashing converts data into a **fixed-length value (hash)**.

It is a **one-way process**, meaning it cannot be reversed.

Hashing में data को fixed output में convert किया जाता है जिसे वापस original form में नहीं लाया जा सकता।

---

## 🔐 How Hashing Works

Momo creates a password.

- The system stores its hash instead of the actual password  
- During login, the entered password is hashed and compared  

Even if the database is leaked, attackers cannot see the original password.

---

## ⚠️ When Hashing Fails

If weak hashing algorithms are used:

- Hackers can crack hashes easily  
- Passwords can be exposed  

अगर weak hashing use हो, तो attacker password guess कर सकता है।

---

## 🎯 Interview Tip

Hashing is used for **password storage and data integrity**.

---

## 🚀 Key Takeaways

- One-way process  
- Cannot be reversed  
- Protects passwords
