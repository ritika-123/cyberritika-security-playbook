# 🔐 Asymmetric Encryption in Cybersecurity

## 📌 What is Asymmetric Encryption?

Asymmetric Encryption uses **two keys**:

- Public Key (shared with everyone)  
- Private Key (kept secret)  

Asymmetric Encryption में data encrypt और decrypt करने के लिए दो अलग-अलग keys होती हैं।

---

## 🔐 How Asymmetric Encryption Works

Momo wants to receive secure messages.

- He shares his public key  
- Anyone can encrypt data using it  
- Only Momo can decrypt it using his private key  

Even if someone intercepts the message, they cannot decrypt it without the private key.

---

## ⚠️ When Asymmetric Encryption Fails

If Momo’s private key gets leaked:

- Anyone can decrypt messages  
- His identity can be misused  

अगर private key compromise हो जाए, तो पूरा system unsafe हो जाता है।

---

## 🎯 Interview Tip

Asymmetric encryption solves the **key sharing problem of symmetric encryption**.

---

## 🚀 Key Takeaways

- Uses public and private key  
- More secure but slower  
- Used in secure communication
